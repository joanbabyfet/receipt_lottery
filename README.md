## About
基于 Python3 采集统一发票中奖号码并自动对奖

## Feature

* 采集发票中奖号码, 网站以 https://invoice.etax.nat.gov.tw/index.html 为例
* 引用 BeautifulSoup 库, 实现选择html DOM
* 实现输入号码并自动对奖功能

## Requires
Python 3.11.0  
beautifulsoup4 4.11.1  
requests 2.28.1  

## Usage
```
python main.py
```
